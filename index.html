Circuit — the map's AI

Without a key, the bar uses a built-in offline brain — it can list issues, find unwired pieces, and suggest connections from its knowledge of this map. Paste an Anthropic API key to switch on Claude mode for real reasoning about your system. The key is stored only in this browser, never inside the file.
Circuit's personality & instructions live in lagbot-map-ai.md next to this file — edit that file to change how it thinks. Suggested connections always wait for your approval before being drawn.

⚔ Hackathon

Red-team the architecture

A self-audit: the agent picks a random angle of attack, walks your documented system looking for weak points, then shows its steps and what blocked it. It threat-models the architecture on this map — it does not touch any live system. Findings come with defensive fixes you can apply.
Improve the Circuit

Ask the Circuit to change itself

Describe a change to this app's own code (a new button, a fix, a tweak). A sandboxed coding agent makes it and opens a pull request — nothing goes live until your team reviews and merges it. That merge is the approval.
Export

What do you want to export?

The agent brief is a Markdown work order for a coding agent: every change the team drew on this map since the baseline — new components, new connections, removals, wireframe edits, open gaps — written as instructions, with the signed change log. The JSON backup is the full map for ⤒ Import.
Team sync

Share this map with your team

Connect a Supabase project and everyone who opens this file sees the same map, live — every change syncs within a second, signed with the editor's name. One-time setup: create a free project at supabase.com, run the SQL from lagbot-map-team-sync.sql (next to this file) in its SQL editor, then paste the project's URL and anon key below.
⚠ If the team already has a shared map in this project, connecting will replace what's on this screen with the shared version. Use ⤓ Export first if you want a backup.

How this map works

Lagbot · System Circuit

Click nodeSpotlights it with everything wired to it; the inspector opens for editing.
Drag nodeMoves it — connections follow. Layout saves automatically.
Double-click pageOpens its app wireframe. Tap an element to light its circuit — ↗ what it calls forward, ↘ what feeds it back from the backend. Elements marked ⇢: tap twice to follow the flow to the next screen or page. Chips above the phone switch screens; ‹ goes back.
+ NodeAdds a new branch: pick a name, type and layer, then drag into place.
⌁ ConnectClick a source node, then a target node, to draw a new connection. Esc cancels.
Click a lineSelects the connection — relabel, flag as gap/planned, or delete it.
⚠ Gap reportSelect any flagged node — the inspector explains what's missing, recommends additions you can add with one tap, and how to fix it.
✦ AI barAsk the map anything from the bar at the bottom. The offline brain works instantly; add an Anthropic API key in ⚙ for Claude-powered reasoning. Suggested lines and nodes always wait for your approval before being drawn.
≡ LogEvery change is signed — each page load asks you to pick your profile (Andre / David / Joseph), then logs who · what · date · time. View or switch profile via ≡ Log.
☁ SyncConnect a Supabase project to share one live map with your whole team — changes sync within a second, signed with each editor's name. One-time setup SQL is in lagbot-map-team-sync.sql.
⌗ OrganiseSnaps every node back into its lane in flow order. Connections are untouched — only positions move.
✦ Trace a flowAsk the AI bar "how do conversations flow?" (or sale / receipt / token / withdrawal, or "from X to Y") and it lights the path and numbers the lines 1·2·3 in order.
✦ Show skillsCircuit backs its answers with action chips — 📍 locate a node, ✦ highlight a set, ▣ open a wireframe, ▶ preview a flow animated node-by-node. The first auto-plays so it shows exactly what it's explaining.
▶ Preview a changeEvery AI suggestion has a Preview button — it animates the new line/node onto the map (or opens the screen) so you see the change before approving.
✦ AI edits UIIn Claude mode, Circuit can move, restyle, retext, rewire or remove existing components — ask it (e.g. "move the skills grid above the toggle"). It reads the open screen (see 👁 View) to match the real layout; every edit is still a proposal you Preview + Approve.
👁 ViewIn a wireframe, 👁 View shows the screen exactly as Circuit reads it — the render it checks your UI edits against.
Reuse componentsIn ✎ edit, each component can be Duplicated, Copied, or Moved to another screen; ▤ Library drops in ready-made components in the app's style; ⧉ Paste reuses a copied one anywhere.
⚔ HackathonA red-team self-audit. It picks a random angle, strobes through your architecture hunting for weak points, then shows its steps, what blocked it, and defensive fixes you can apply with one tap. ≡ History keeps every run.
Edit wireframe✎ turns on edit mode: add screens with the ＋ chip, tap a component to edit every property (text, badges, amounts, stats…), drag components to reorder — they snap between rows, or side-by-side when dropped on a row's right edge. + Add element places new components (bubbles, buttons, text, stats…), then wire them to backend nodes.
SearchType to dim everything except matching nodes.
KeysF fit · + / − zoom · Del removes selection · Esc closes / cancels.
SavingEvery change persists automatically. ⤓ Export offers a coding-agent brief (.md work order of all team changes) or a JSON backup; ⤒ Import restores a JSON backup; Reset restores the original Lagbot map.
